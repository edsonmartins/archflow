# archflow Vue Example

Exemplo de integração do Web Component archflow com Vue 3.

## Funcionalidades

- Uso do `<archflow-designer>` em Vue
- Chat com streaming de respostas
- Integração com API do archflow

## Instalar

```bash
npm install
```

## Executar

```bash
npm run dev
```

Acesse: http://localhost:5173
