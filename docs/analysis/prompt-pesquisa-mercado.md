# Prompt para Pesquisa de Mercado - archflow

Copie e use este prompt para pesquisar produtos concorrentes e tendências do mercado:

---

## Prompt para Pesquisa

```
Preciso de uma análise detalhada do mercado de plataformas/workflows para IA em 2025-2026. Foco em:
1. Java/AI workflow orchestration platforms
2. Visual AI builders (tipo LangFlow, Flowise)
3. Enterprise AI platforms com workflow capabilities

Para cada produto encontrado, preciso:
- Nome e empresa
- Stack tecnológica (Java, Python, Node.js?)
- Tem visual workflow designer? Sim/Não
- Tem plugin system? Sim/Não
- Enterprise-ready? (métricas, auditoria, SSO, etc.)
- Preço/modelo de negócio
- Pontos fortes e fracos
- Data da última atualização

Produtos para investigar a fundo:
- LangFlow
- FlowiseAI
- LangChain4j (foco em recursos de orquestração)
- Spring AI (foco em recursos de orquestração)
- Temporal (para AI workflows)
- Camunda 8 (para AI workflows)
- EasyWorkflow para LangChain4j
- LangGraph4j
- Outros que encontrar

Também preciso:
- Tendências do mercado AI orchestration para 2025-2026
- TAM (Total Addressable Market) para AI workflow platforms
- Principais use cases empresariais
- O que está faltando no mercado (gaps)

Retorne uma análise estruturada com:
1. Mapa competitivo detalhado
2. Matriz de comparação
3. Gaps de mercado identificados
4. Oportunidades para um produto Java-nativo
```

## Prompt Alternativo (Mais Focado)

```
Pesquise especificamente sobre:

1. **Visual AI Builders em 2025-2026**
   - Quais são os principais?
   - Qual stack tecnológica usam?
   - Existe algum Java-nativo?
   - Quais recursos enterprise têm?

2. **Enterprise AI Workflow Platforms**
   - Como empresas orquestram workflows de IA em produção?
   - Que ferramentas usam?
   - Que problemas enfrentam?

3. **Java AI Ecosystem**
   - Quais frameworks Java para IA estão ativos?
   - Qual o mais usado em 2025?
   - O que está faltando?

Para cada ferramenta encontrada, crie um perfil com:
- Nome/URL
- Ano de criação
- Stars no GitHub (se open source)
- Último commit
- Linguagem principal
- Tem visual designer?
- Tem plugin system?
- Tem features enterprise?
- Preço

Por fim, responda:
- Existe espaço para um produto Java-nativo com visual designer + plugin system + enterprise features?
- Quais seriam os diferenciais competitivos?
```

## Dicas para Pesquisa Eficaz

1. **Comece amplo, depois foque**: Primeiro mapeie o cenário geral, depois deep dive em cada produto

2. **Verifique datas**: AI evolui rápido - priorize conteúdo de 2025-2026

3. **Olhe atividade do projeto**: Último commit, issues abertas, discussões recentes

4. **Multi-fonte**: Cross-reference informações de várias fontes

5. **Foco em gaps**: Anote o que NÃO existe (pode ser oportunidade)
