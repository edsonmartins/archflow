# LinkedIn Post — archflow v1.0.0 Launch

---

🚀 Excited to announce **archflow v1.0.0** — an open-source Java framework for building AI agent workflows!

After months of development, archflow is ready for production. Here's what makes it unique:

🎨 **Visual Workflow Designer** — Drag-and-drop interface for designing AI workflows, available as a React component or standalone Web Component

🔗 **LangChain4j Native** — Deep integration with 15+ LLM providers (OpenAI, Anthropic, Google, Mistral, Ollama, and more)

🔌 **MCP Support** — Model Context Protocol for standardized tool access across AI providers

🏢 **Enterprise Ready** — JWT/RBAC authentication, OpenTelemetry tracing, Prometheus metrics, audit logging, and two-level caching

🧩 **Plugin Architecture** — Extend with custom tools, assistants, and agents. Built-in marketplace for discovery and installation.

💬 **Conversational** — Suspend/resume workflows with dynamic forms for human-in-the-loop scenarios

Built with: Java 17+, Spring Boot 3.3, React 19, TypeScript

540+ tests | Full CI/CD | Docker & Kubernetes ready

🔗 GitHub: https://github.com/edsonmartins/archflow

If you're building AI applications in Java, give archflow a try and let me know what you think!
